Programme Overview

Jio Institute's M.Tech in Artificial Intelligence & Data Science is a rigorous two-year programme that equips students with cutting-edge knowledge and practical skills in AI, machine learning, data engineering, and analytics. Designed in collaboration with global academic and industry partners, the programme prepares graduates to lead innovation in a data-driven world.

Curriculum Overview

Mathematical Foundation

Build the quantitative backbone required for advanced AI and data science. This module strengthens your understanding of the mathematical principles that power modern algorithms and machine learning models.

Probability Statistics Linear Algebra ML Intuition

Python Programming

Master Python for data science, covering libraries, scripting, and application development for AI solutions.

Data Structures and Algorithms

Learn essential data structures and algorithms for efficient data processing and problem solving.

Databases & Data Warehouses

Understand relational databases, NoSQL, and data warehousing techniques for managing large-scale datasets.

Optimization for Business

Apply optimization techniques to solve complex business problems using mathematical modeling and algorithms.

Data Visualization

Create compelling visual representations of data using industry-standard tools and best practices.

Time Series Analysis

Analyze temporal data patterns and build forecasting models for real-world applications.
